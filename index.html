01. The Monolith Trap:
ERPNext Reality Check

Metplast currently operates on ERPNext—a generic, open-source platform. While it provides baseline accounting, it is structurally flawed when forced to manage complex field logistics, mobile operations, and custom manufacturing intelligence.

Where ERPNext Falls Short (Documented Gaps)

The system fundamentally fails when extended to physical field operations:

  • Mobile App Officially Deprecated: The Frappe team formally announced deprecation of native mobile apps in Dec 2021. The official position is 'use the mobile browser.' This means no native push notifications, no offline capability, and no reliable QR scanning.
  • Gantt Chart Bugs: Documented GitHub Issue #19348 and #40818 show the Gantt chart handles snap incorrectly, and moving tasks can recursively reset dependent timelines.
  • 2,000+ Open GitHub Issues: Bug fixes are prioritized by the core team on their schedule; businesses cannot escalate critical operational bugs.
  • Updation Fragility: ERPNext major versions (v14 → v15 → v16) change database structures significantly. Version upgrades consistently break custom modifications, leaving you locked into outdated versions.
  • No Purpose-Built Service App: There is no mobile-first, offline-capable app for field technicians to receive jobs, scan parts, and capture digital customer sign-offs.

Head-to-Head Architectural Comparison

Operational Capability Vanilla ERPNext Metplast Custom OS (Next.js)
User Interface (UI) Generic UI with 50+ irrelevant tabs causing severe cognitive fatigue for operators. Role-Based Micro-Dashboards. Operators only see what they need (e.g., Floor Manager sees only Work Orders & OEE).
Mobile Strategy Deprecated native apps. Forces field workers to use a clunky mobile web browser. Native React Native Android APKs. Fast, intuitive, big-button UI designed for rugged field conditions.
Offline Field Resilience Fails immediately if 4G connection drops. Data loss occurs. SQLite Queue Caching. Works flawlessly offline in basements/remote sites, syncing automatically upon reconnection.
Accounting & Auditing Traps financial data inside its own proprietary ledger system. Tally-Ready Export. Instantly export invoices/ledgers in CSV/XML specifically formatted for your CA's Tally software.
Developer Availability Frappe/Python developers are scarce, expensive, and create "Hit-by-a-bus" lock-in. Next.js / Node.js. Built on the world's most ubiquitous, easily-sourceable JavaScript framework.

03. Unified Module
Specifications

A breakdown of the 6 core systems. Unlike generic ERPs, these modules are heavily interlinked to automate data flow across departments, drastically extending operational capabilities.

The Unified Data Sync & Role-Based Dashboards

Everything is connected. Nothing is cluttered.

In generic ERPs, every user sees everything, leading to massive UI fatigue. We employ Role-Specific Micro-Dashboards. The CFO's dashboard only surfaces Cash Flow and Receivables. The Floor Manager's dashboard only shows Machine OEE and Active Work Orders.

Yet, underneath, the data is flawlessly synced: a Telegram lead conversion instantly generates a quotation, which calculates a BOM, reserves inventory, and plots a Gantt timeline. The workflow is automated, and the user interface remains perfectly clean.

04. Specialisation &
Differentiators

Deep dives into the bespoke workflow engines that provide Metplast with an insurmountable operational advantage.

Telegram Bot Lead Routing

Drops lead response time from hours to under 2 minutes.

Lead arrives and system dispatches a formatted Telegram message to the admin group with inline buttons [Assign to Ravi]. Admin taps, and the salesperson receives an instant notification with a deep link to the CRM record. If unassigned for 30 mins, the bot escalates.

Cage Layout Auto-Generator

Turns 3 hours of manual quotation work into 5 minutes.

Sales user inputs Shed Length × Width, Floors, Target Capacity. System calculates optimal rows, walkway clearances, feed line positioning, and generates a visual SVG/PDF diagram automatically embedded into the Metplast-branded quote.

QR Material Lifecycle Tracking

Eliminates material disputes and establishes a chain-of-custody.

Stage 1: Generate QR at production. Stage 2: Warehouse scans to log loading to vehicle. Stage 3: Field team scans at customer site to log GPS-verified offloading with digital signature. Management views complete audit trail.

Task Assignment & Feedback Loop

Every task becomes a documented, accountable thread.

Assignee receives a task and must respond: ACCEPT, FLAG CONCERN, PROPOSE ALTERNATE TIMELINE, or REQUEST REASSIGNMENT. Eliminates verbal "I thought you were doing it" situations with full audit trails.
